🔒

Fully offline

No telemetry. No phone-home. No data sent anywhere. Both tools work air-gapped. The only network call is the binary download at install time.

✅

Signed and verified

Release binaries are signed with cosign (keyless, Sigstore). SLSA provenance and SPDX SBOM published with every release. Verify the checksum before running.

🔓

Open source (Apache-2.0)

Full source on GitHub. Deterministic builds. No obfuscation. Security tools that aren't auditable should not be trusted.

⚡

Read-only, always

These tools never modify your configs, your log files, or anything else on disk. They read, analyze, and report.
