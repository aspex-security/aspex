# MCP006: Secrets in config env

| Field | Value |
|---|---|
| ID | MCP006 |
| Severity | CRITICAL |
| Mapping | OWASP LLM06 |
| Status | Active (v0.1) |

## What it detects

Env key names in MCP server config that match known secret patterns: AWS keys (AKIA..., AWS_SECRET_ACCESS_KEY), GitHub tokens (ghp_, GH_TOKEN), OpenAI/Anthropic/Stripe/Sendgrid API keys, and generic _API_KEY, _SECRET_KEY, _PASSWORD, _TOKEN suffixes.

Note: only key names are checked, never values. The values are not read.

## Why it matters

MCP client configs (claude_desktop_config.json, .cursor/mcp.json) are plaintext JSON files on disk. Any secret pasted into the env block is readable by any process with filesystem access and may be committed to version control.

## Fix

Move secrets to a vault, OS keychain, or environment variable set outside the config file. Reference them via the shell environment rather than hardcoding in the config.
