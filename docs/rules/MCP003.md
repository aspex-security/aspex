# MCP003: Dangerous capability: shell/exec

| Field | Value |
|---|---|
| ID | MCP003 |
| Severity | CRITICAL |
| Mapping | OWASP LLM08 |
| Status | Active (v0.1) |

## What it detects

Tools named or described in ways that imply shell command execution: run_command, execute_command, shell, exec, bash, sh, eval, run_script.

## Why it matters

Shell execution is the highest-risk capability an MCP server can expose. Combined with prompt injection or confused deputy attacks, it enables arbitrary code execution on the developer's machine.

## Fix

Remove the shell execution grant, or restrict it to an explicit allow-list of safe commands. Use a more specific tool instead of general shell access.
