# MCP001: Prompt injection in tool description

| Field | Value |
|---|---|
| ID | MCP001 |
| Severity | HIGH |
| Mapping | OWASP LLM01 |
| Status | Active (v0.1) |

## What it detects

Imperative patterns in a tool description that attempt to hijack the model: "ignore previous instructions", "you must act as", "system prompt", hidden Unicode control characters (zero-width joiners, formatting characters) that carry invisible instructions.

## Why it matters

The model reads tool descriptions before deciding which tool to call. A malicious server can embed instructions that redirect the model to exfiltrate data, call other tools, or override the user's intent.

## Fix

Remove or rewrite the tool description. If the server is third-party, remove it from your config.
